## 🚀 特性

- Vue3/Element Plus/Pinia
- ESLint/Prettier集成
- husky/commitlint集成
- 支持多页
- 组件自动注册
- Mock
- TailwindCSS集成
- 暗黑模式/色弱模式/灰色模式
- 支持主题色换肤
- svg雪碧图与自动下载iconfont方案
- 内置常见解决方案（平滑滚动/复制粘贴/断网检测/全屏/快捷键/功能指引/水印/图表等）
