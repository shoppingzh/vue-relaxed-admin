**index.scss**

入口文件，将每个单页都需要的最基础的样式文件统一引入
注：该文件不一定引入所有样式文件，有一些特有的样式，请在对应单页内引入


**global.css**

全局样式，例如全局设置字体大小、字体颜色、滚动条样式等。


**defaults.css**

html的默认标签样式修改，如h1/h2/h3/h4/h5/h6与a标签等样式覆盖。

**theme.scss**

主题相关的样式变量配置。

**tailwind.css**

TailwindCSS

**element-plus.css**

主要是对element-plus的样式进行覆盖，如修改主题色等。
